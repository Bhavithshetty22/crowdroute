# API Roadmap

## Planned Integrations
1. **Google Maps Directions API**: Evolving from raw path calculation to live transit times outside the venue.
2. **Google Places API**: Real-time food vendor information wrapping for the Dashboard modules.
3. **Vertex AI Multimodal**: Transitioning from pure text fallbacks to visual recognition of queues.
4. **Cloud Run / Fastify**: Future replacement of the baseline container to an enterprise microservices mesh.
