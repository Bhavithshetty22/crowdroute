# CrowdPilot AI: Cloud Architecture

## Overview
CrowdPilot AI is designed around a serverless, highly-scalable cloud architecture meant to deploy seamlessly onto Google Cloud Platform. 

## Key Integrations (Roadmap & Ready)
- **Firebase Auth**: Secure, native identity management.
- **Firestore**: Real-time NOSQL database for user states and crowd tracking updates.
- **Firebase Analytics & BigQuery**: Every user interaction, from routing to emergency module toggles, is prepared to be ingested by Firebase and streamed into BigQuery for post-game predictive analytics.
- **Cloud Functions**: Designed to securely host proprietary Vertex AI pipeline requests.
- **Vertex AI & Gemini**: LLM capabilities are fundamentally integrated to provide dynamic natural-language interpretation on venue constraints.
- **Google Maps API**: Context-aware routing hooks into official mapping primitives.
